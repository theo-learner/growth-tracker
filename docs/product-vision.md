# 성장 트래커 — 제품 비전 문서

> 작성: 2026-02-27 | 출처: 창업자 초기 아이디어 전사문 분석

---

## 창업 배경 (원문 인용)

> "하루하루 시안이의 진행이 얼만큼 개량됐는지 궁금해.
> 엑슬러 검사를 하지 않아도 얘의 지능이 어느 정도 될지 예측해주는 서비스가 있으면 좋겠어."

부모(주양육자)가 이미 아이의 강점·약점을 직관적으로 알고 있다.
다만 **전문 용어**를 모를 뿐이고, **정량화·시각화** 도구가 없다.
이 앱의 역할은 그 관찰을 **구조화·해석·예측**해주는 것이다.

---

## 핵심 가치 루프

```
부모의 일상 관찰 (인풋)
    ↓
활동 기록 (텍스트 / 사진 / 자연어)
    ↓
AI 발달 분석 + 점수 추적
    ↓
"얼마나 성장했는지" 가시화 (아웃풋)
    ↓
맞춤 교구·활동 추천 → 더 좋은 인풋 제공
```

---

## 창업자가 꿈꾼 구체 기능

### 1. 활동별 정량 추적 ⭐ (핵심)
> "72조각 50분 → 48분 → 108조각 도전"

같은 활동의 **난이도 + 완성 시간**을 누적 기록하여
"이번 달 퍼즐 평균 완성 시간이 지난달 대비 -8분" 같은 정량적 성장을 보여줌.

### 2. 사진 기반 AI 분석
> "채찍PT한테 사진 찍어 보내서 얘 인지 레벨 물어봐"

업로드된 사진을 Claude Vision으로 분석해 활동 맥락·인지 수준 자동 추출.

### 3. 쉬운 언어 온보딩
> "전문 용어를 모를 뿐이지, 주양육자 엄마는 다 알아. 그걸 풀어서 설명해주면 된다"

K-WPPSI 용어 대신 "새로운 것에 금세 적응해요 / 조금 시간이 걸려요" 형태로 기질·강점 입력.

### 4. 장기 AI 맥락 (GPT 메모리 개념)
> "GPT한테 시안이에 대해 1년 동안 학습을 시킨 거야. 메모리에 다 쌓여있으니까"

매 분석 시 전체 누적 기록 + 기질 + 기존 검사 결과를 AI에게 전달해 개인화된 인사이트 생성.

### 5. "상위 몇 퍼센트" 표현
> "이 정도 책을 읽고 이해한다면 상위 몇 퍼센트 수준이다"

점수(40~100)를 또래 비교 백분위로 번역해 부모에게 직관적으로 전달.

### 6. 교구 추천 (현재 부분 구현)
> "쿠팡에서 유사한 교구를 찾는 노동을 AI가 대신해줬으면"

AI가 추천하는 교구가 실제로 없으면 유사 기능 제품으로 자동 대체.

### 7. 비즈니스 모델
- 무료: 사진 10장, 기본 분석
- 유료 $10/월: 무제한 사진 + 동영상 분석
- 배포: 앱스토어 / 토스 미니앱
